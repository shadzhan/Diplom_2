# Diplom_2
Дипломная работа задание 2
Проект тестирования API Stellar Burgers
Этот проект предназначен для автоматизированного тестирования API сервиса Stellar Burgers с использованием pytest, requests и allure-pytest. В рамках проекта реализованы тесты для проверки основных эндпоинтов, таких как получение ингредиентов, создание заказов, регистрация и авторизация пользователей, сброс пароля и получение данных о заказах.

Структура проекта
your_project/
│
├── requirements.txt          # Зависимости проекта
├── conftest.py               # Фикстуры pytest (в корне)
└── tests/                    # Тестовые сценарии
    ├── data.py               # Тестовые данные
    ├── test_ingredients.py   # Тесты для получения ингредиентов
    ├── test_order_creation.py# Тесты для создания заказов
    ├── test_password_reset.py# Тесты для сброса пароля
    ├── test_user_registration.py # Тесты для регистрации
    ├── test_user_login.py     # Тесты для логина
    ├── test_user_update.py    # Тесты для обновления данных пользователя
    └── test_get_orders.py     # Тесты для получения заказов
Установка зависимостей
Для запуска тестов необходимо установить все зависимости:

bash
Copy code
pip install -r requirements.txt
Файл requirements.txt содержит:

pytest
requests
allure-pytest
Запуск тестов
Запустите тесты с помощью pytest и сохраните результаты для отчёта:
bash
Copy code
pytest --alluredir=allure-results
Откройте отчёт в браузере:
bash
Copy code
allure serve allure-results
Структура и использование
conftest.py — содержит фикстуры для генерации уникальных данных, авторизации и хранения токенов.
data.py — содержит тестовые данные, такие как ингредиенты, тестовые пользователи.
Тестовые сценарии — реализованы в отдельных файлах по функциональности, используют фикстуры и данные из data.py.
Особенности реализации
Тесты независимы и используют фикстуры для подготовки данных.
Проверяется как тело ответа, так и статус код.
В тестах реализована проверка различных сценариев, включая успешные и ошибочные.
Отчёт в Allure помогает визуализировать результаты тестирования.
Внесение изменений и расширение
Для добавления новых тестов создавайте новые файлы в папке tests/.
Используйте фикстуры из conftest.py.
Для тестовых данных редактируйте data.py.
